# Ticket-Management

install database "ticketmanage.sql" before use clone git command

clone git

    git clone https://github.com/yitaraZ/Ticket-Management.git
    
install dependency

    cd Ticket-Management
    cd backend
    npm i
    cd ..
    cd my-todo-react
    npm i
    
run backend sql 

    cd backend
    npm run dev
    
run frontend

    cd my-todo-react
    npm run dev
